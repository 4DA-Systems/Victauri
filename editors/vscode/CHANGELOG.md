# Changelog

## 0.1.0 (2026-05-13)

Initial release.

- Activity bar with App State, DOM Explorer, and IPC Log tree views
- Live polling of windows, memory, DOM, IPC traffic, and diagnostics
- Screenshot panel with refresh and save-to-disk
- CodeLens on `#[tauri::command]` for test generation
- Status bar connection indicator
- Auto-connect when Tauri project detected in workspace
- Port discovery via `victauri.port` temp file
- Bearer token auth support
- Evaluate JS in the webview via input box
- Run diagnostics for compatibility checks
- Right-click DOM elements to click, highlight, inspect styles, copy ref IDs, or generate test code
- Accessibility audit command (WCAG checks)
- Performance metrics command (navigation timing, JS heap, DOM stats)
- Clear highlights command
